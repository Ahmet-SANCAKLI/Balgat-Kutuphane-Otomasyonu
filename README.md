# 📚 Balgat Kütüphane Otomasyonu

Bu proje, yerel bir kütüphanenin kitap takibi, üye yönetimi ve ödünç alma işlemlerini dijitalleştirmek amacıyla geliştirilmiş bir **Kütüphane Yönetim Sistemi**dir.

## 🚀 Özellikler
- **Kitap Yönetimi:** Kitap ekleme, silme, güncelleme ve listeleme.
- **Üye Takibi:** Kütüphane üyelerinin kayıtlarını tutma ve yönetme.
- **Ödünç İşlemleri:** Hangi kitabın hangi üyede olduğunu, teslim tarihlerini takip etme.
- **Veritabanı Desteği:** Verilerin güvenli ve hızlı bir şekilde saklanması için **SQLite** entegrasyonu.

## 🛠️ Kullanılan Teknolojiler
- **Dil:** Python 3.11.9
- **Veritabanı:** SQLite3
- **Mimari:** Modüler yapı (`models.py`, `actions.py`, `app.py`)
